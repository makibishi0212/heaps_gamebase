# GameBase

### resolve haxelib dependency

```
haxelib git deepnightLibs https://github.com/deepnight/deepnightLibs.git
haxelib git castle https://github.com/ncannasse/castle.git
```

### compile

```
haxe hl.sdl.hxml
```

(DirectX compile is not work Mac OSX)

##### for web

```
haxe js.hxml
```

### running

```
hl bin/client.hl
```
